# Roadmap

This page tracks the next topics planned for the repository.

The goal is not to cover everything in ML. The goal is to expand the highest-signal topics for:

- ML / LLM interviews
- applied research interviews
- AI systems and serving interviews

## In Progress Direction

- make the repository easier to browse publicly
- add more cross-links between notes
- keep the notes compact while increasing depth on high-frequency topics

## Planned Notes

### Core LLM

- attention variants and kernel-level intuition
- MoE routing, load balancing, and serving tradeoffs
- quantization deep dive
- KV cache architecture and serving tradeoffs

### RL and Post-Training

- PPO from objective to implementation
- GAE derivation and intuition
- reward modeling failure modes
- online RL for LLM reasoning

### Agent and Reasoning

- inference-time scaling
- planner vs executor architectures
- multi-agent systems
- browser / code / tool-use agent patterns

### Systems

- vLLM walkthrough
- paged attention intuition
- continuous batching and admission control
- prefix cache design tradeoffs

## Possible Future Additions

- English summary pages for the highest-traffic notes
- interview question banks by topic
- diagrams for serving systems and post-training pipelines
- comparison notes across popular open-source inference stacks

## Selection Principle

New notes are added when they satisfy at least one of these:

- high interview frequency
- high confusion rate across similar concepts
- strong system design or implementation value
- useful as a compact review note before interviews
